toobox_signal - signal processing related functions.

This toolbox contains functions related to signal processing, including 
* 1D and 2D filtering, filter design.
* powerful various arithmetic coding scheme.
* histogram computation and matching.
* vector quantization using kmeans.
* Walsh and DCT transforms.
* images loading and generation using 1/f noise.
* 1D signals loading and generation.
* Radial basis functions.
* Various helpers functions.
