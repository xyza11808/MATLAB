Waveform Amplitude Distribution Histogram

Email: sigtool@kcl.ac.uk

 

Author: Malcolm Lidierth

Updated: August 2008

Method Summary

Forms the frequency distribution of the amplitudes for a waveform channel

 

Options

Channel A (Waveforms)

Select one or more waveform channels to use as the trigger. Both continuous and episodically sampled channels may be analyzed. For episodic data, only valid epochs will be used.

Number of bins

Select the number of bins in tha range data mimimum to data maximum for each channel

Start & Stop

Only data between the specified times will be processed.
