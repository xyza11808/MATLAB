Select Subchannel

Email: sigtool@kcl.ac.uk

 

Author: Malcolm Lidierth

Updated: February 2008

When a file contains multiplexed waveform channels, you can select which subchannel to analyze from each channel using the Channel->Select Subchannels menu. This allows the current subchannel setting for each channel to be changed. By default, the current subchannel is 1.

 

All relevant commands from the Channel menu (Decimate, Digital Filter, Copy To...) respect the current subchannel setting. Those in the Waveform menu may or may not. You can still apply analysis routines that do not support multiplexed data to each subchannel, but you must first extract the relevant subchannel using the Channel -> Copy To Temporary Channel command. This copies the data from the current subchannel of the selected channel to a new, non-multiplexed, channel.

By default, subchannel 1 is set as the current subchannel. You can alter this setting by selecting from the list of multiplexed channel and setting a new value in the Subchannel field.

Reset All returns the current subchannel to 1 on all multiplexed channels.

'Apply to all open files' is not activated in this menu.
