To install sigTOOL, first extract the main file to the required base folder.
Note that this must end as "....\sigTOOL".
If your archive extractor adds the version number to the folder name,
rename it after extracting the files.

The end-user will need write-access to this folder.


For further steps in the installation, see the "Installing sigTOOL" PDF file.


User and Programming Guides are included in the ...sigTOOL\documentation folder.
